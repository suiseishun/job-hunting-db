# 就活企業データベース

自作の就活管理ツールです。
複数の企業データ（年収、従業員数、特徴など）を一覧比較し、自分の選考状況をローカルストレージで管理できます。

## 主な機能
- 企業検索・フィルタリング: 業界や業種、フリーワードでの絞り込みが可能です。
- 選考状況管理: 「インターン」「本選考」「内定」などのステータスを色別で管理できます。
- 年収・従業員数ソート: 平均年収や新卒初任給での並び替えに対応しています。
- データ保存: 入力したメモや選考状況はブラウザ（Local Storage）に保存されます。

## 技術スタック
- HTML5 / CSS3 (Neumorphism UI)
- JavaScript (Vanilla JS)

## 使用方法
1. `index.html` をブラウザで開くか、GitHub PagesのURLにアクセスしてください。
2. 興味のある企業をお気に入りに登録したり、選考状況を更新して管理してください。
